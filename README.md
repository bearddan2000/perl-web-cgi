# perl-web-cgi

## Description
A POC perl cgi script.

## Tech stack
- perl 5

## Docker stack
- nginx:latest
- perl:latest

## To run
`sudo ./install.sh -u`
Availble at http://localhost

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
